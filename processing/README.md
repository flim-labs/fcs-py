
# How to run


#### pyo3 - root folder

```bash
cd processing/fcs-pyo3-wrapper
```

#### pyo3 - build release 

```bash
maturin build 
```

#### pyo3 - build develop 

```bash
maturin develop 
```

<hr>

#### Rust - root folder

```bash
cd processing/fcs-rust
```

#### Rust - build release 

```bash
cargo build --release
```

#### Rust - run  

```bash
cargo run --release
```
